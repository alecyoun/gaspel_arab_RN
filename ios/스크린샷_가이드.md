# App Store 스크린샷 생성 가이드

## 필요한 크기
- **세로 (Portrait)**: 1242 × 2688px 또는 1284 × 2778px
- **가로 (Landscape)**: 2688 × 1242px 또는 2778 × 1284px

## 방법 1: 자동 스크립트 사용 (권장)

터미널에서 다음 명령어를 실행하세요:

```bash
cd /Users/dora/Documents/gaspel_arab_RN/ios
./take-screenshots.sh
```

스크립트가 자동으로:
1. 시뮬레이터를 실행합니다
2. 스크린샷을 찍습니다
3. 올바른 크기로 조정합니다
4. 데스크톱에 저장합니다

## 방법 2: 수동으로 스크린샷 찍기

### 1단계: 시뮬레이터 실행
1. Xcode를 엽니다
2. **Window** → **Devices and Simulators** (Shift + Cmd + 2)
3. **iPhone 14 Pro Max** 또는 **iPhone 15 Pro Max** 선택
4. 시뮬레이터를 실행합니다

### 2단계: 앱 실행
1. Xcode에서 프로젝트를 엽니다
2. 시뮬레이터를 선택합니다 (상단 바에서)
3. **Run** 버튼을 클릭하거나 Cmd + R을 누릅니다
4. 앱이 실행될 때까지 기다립니다

### 3단계: 스크린샷 찍기

#### 세로 모드 스크린샷
1. 시뮬레이터에서 원하는 화면으로 이동합니다
2. **File** → **New Screen Recording** 또는 **Cmd + S**를 누릅니다
3. 또는 터미널에서:
   ```bash
   xcrun simctl io booted screenshot ~/Desktop/screenshot_portrait.png
   ```

#### 가로 모드 스크린샷
1. 시뮬레이터를 가로 모드로 회전합니다 (Cmd + 좌/우 화살표)
2. 원하는 화면으로 이동합니다
3. 스크린샷을 찍습니다:
   ```bash
   xcrun simctl io booted screenshot ~/Desktop/screenshot_landscape.png
   ```

### 4단계: 이미지 크기 조정

스크린샷이 올바른 크기가 아닐 수 있습니다. 다음 명령어로 크기를 조정하세요:

```bash
# 세로 모드 (1284 × 2778px)
sips -z 2778 1284 ~/Desktop/screenshot_portrait.png --out ~/Desktop/screenshot_portrait_1284x2778.png

# 가로 모드 (2778 × 1284px)
sips -z 1284 2778 ~/Desktop/screenshot_landscape.png --out ~/Desktop/screenshot_landscape_2778x1284.png
```

또는 **미리보기(Preview)** 앱에서:
1. 이미지를 엽니다
2. **도구** → **크기 조정**
3. 원하는 크기로 설정합니다

## 추천 스크린샷 구성

App Store에 업로드할 스크린샷은 다음을 포함하는 것이 좋습니다:

1. **메인 화면** (찬송가 목록)
2. **찬송가 상세 화면** (가사/악보)
3. **음악 플레이어 화면** (있는 경우)
4. **검색 화면** (있는 경우)
5. **설정 화면** (있는 경우)

## 주의사항

- 스크린샷은 실제 앱 화면과 일치해야 합니다
- 상태 바(시간, 배터리 등)는 포함되어도 됩니다
- 최대 10개의 스크린샷을 업로드할 수 있습니다
- 앱 미리보기는 최대 3개까지 가능합니다

## 문제 해결

### 시뮬레이터가 실행되지 않는 경우
```bash
# 시뮬레이터 목록 확인
xcrun simctl list devices

# 특정 기기 부팅
xcrun simctl boot "iPhone 15 Pro Max"
```

### 이미지 크기 조정이 안 되는 경우
- **미리보기(Preview)** 앱 사용
- 온라인 이미지 리사이저 사용 (예: https://www.iloveimg.com/resize-image)
